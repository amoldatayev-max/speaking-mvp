PROJECT

zest mvp

A voice-first AI speaking application designed to help users practice speaking through natural conversation with an AI speaking coach.

CORE IDEA

The product helps users improve spoken communication by speaking with an AI conversation partner.

The interaction should feel natural, calm, and supportive.

CORE INTERACTION LOOP

User speaks  
→ speech-to-text converts speech into text  
→ AI understands the message  
→ AI generates a response  
→ text-to-speech converts the response into audio  
→ audio reply is played

PRODUCT PRINCIPLE

1% improvement → 99% results.

The AI should focus on one small improvement at a time.

The goal is to keep the user speaking.

INCLUSIVITY PRINCIPLE

The system must remain inclusive and accessible.

The interface should be simple and voice-first.

Users should feel safe practicing speaking.
